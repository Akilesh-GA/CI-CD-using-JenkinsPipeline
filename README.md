# 🚀 Jenkins CI/CD Pipeline on AWS

## 📌 Overview
This project demonstrates a complete CI/CD pipeline using Jenkins deployed on AWS EC2.

## ⚙️ Tech Stack
- Java
- Maven
- Jenkins
- AWS EC2
- Tomcat

## 🔄 Pipeline Stages
1. Checkout (GitHub)
2. Build (Maven)
3. Test
4. Deploy (Tomcat)

## 🚀 Setup Steps
1. Launch EC2 instance
2. Install Jenkins
3. Install Tomcat
4. Configure Pipeline
5. Connect GitHub repo
